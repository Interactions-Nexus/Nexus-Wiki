📱 移动钱包

移动（手机）钱包处于内测阶段，将在 5.1 发布后推出。

任何区块链平台要想取得远程成功，就需要让普通人能够轻松使用该技术。 Nexus 麟社通过真正去中心化的移动钱包和签名链实现了这一点。

移动钱包将区块链置于每个人的手中，将其与免费交易和使用 API 应用程式界面构建 dapps 去中心化应用的便利性相结合，开发人员可以使用这些 API 应用程式界面来设计 DApps 去中心化应用，这反过来将为 Nexus 麟社创造实用性。
 
传统上，移动钱包使用由第三方或钱包开发商运行的代理服务器与区块链网络**.** 进行通信，代理服务器充当可信方，并且是有关区块链状态的信息的唯一提供者。 着一方面，Nexus 麟社钱包运行利用相同的桌面内核，通过从网络种子节点下载对等点列表并与多个对等点建立连接，作为精简节点运行。 这允许钱包直接向网络广播交易，并通过检查是否包含区块头来检测确认。 Nexus 麟社移动钱包通过能够验证由多个对等点而不是通过一个中央节点提供的区块头来获得更高的安全性。 对于精通技术的人来说，这也称为简化支付验证。 Simplified Payment Verification.
 

对比传统模式 与 Nexus 麟社模式移动钱包

遵循 Nexus software stack 麟社软件堆栈的原则，新的移动 dapp 去中心化应用将能够与本地钱包进行交互，而不是依赖单独的服务器来获取有关区块链状态的信息。 Nexus 麟社移动钱包也是完全非托管的，这意味着您的 signature chain 签名链保持完全私密。 所有交易都在本地设备上处理和打包，然后广播到网络的其余部分，就像一个完整的节点。

根据需要，Nexus 麟社移动钱包只会存储和处理区块头和用户的签名链，从硬编码到钱包中的第一个 Tritium 麟氚区块的哈希值开始。 对于运行自己的完整节点和某些 API 应用程式界面功能的高级用户，可以使移动钱包建立直接连接。 这使用户能够将繁重的计算工作卸载到他们的桌面节点，例如搜索历史数据或维护区块链的完整副本。
