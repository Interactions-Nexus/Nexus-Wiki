矿机配置

矿机配置是最关键的部分，使用 JSON 格式。 此配置对于 windows 或 linux 是相同的。

注意：确保 NexusMiner 可执行文件和矿工配置位于同一文件夹中

矿工配置存储在 miner.conf 文件中，它应该与 NexusMiner 可执行文件位于同一文件夹中。 在带有 UI 的 Windows 或 Linux 中，您可以创建一个文本文件并保存配置并将文件另存为 miner.conf。

对于 linux CLI，请使用以下命令：

切换到 NexusMIner 可执行文件所在的 Miner 文件夹，如果它是一个不同的文件夹，则相应地更改。

cd Miner

复制

创建 miner.conf 文件。

nano miner.conf

复制

在下面找到 prime 和 hash 矿工的 JSON 配置文件。 复制粘贴并根据设置更改设置。 请记住，每个 GPU 或 FPGA 都必须配置为唯一的工作线程。 还要确保 “wallet_ip”、“port”、“local_ip”、“mining_mode” 和 “pool” 的详细信息正确无误。

对于 Nexus 麟社 prime 质数和 hash 哈希挖矿池，支付地址将是池的标识符，奖励支付将发送到同一地址。 大多数池在支付前都有最低奖励数额。

 

Prime Pool 质数矿池

{
    "version" : 1,
    "wallet_ip" : "primepool.nexus.io",
    "port" : 50000,
    "local_ip" : "auto",
    "mining_mode" : "PRIME",
    "pool" :
    {
        "username" : "<INSERT_HERE NXS ADDRESS>",
        "display_name" : "<INSERT HERE>"
    },
    "stats_printers" :
    [
        {
            "stats_printer" :
            {
                "mode" : "console"
            }
        }
    ],
    "workers" :
    [
        {
            "worker" :
            {
                "id" : "myWorker0",
                "mode" :
                {
                    "hardware" : "gpu",
                    "device" : 0
                }
            }
        }
    ]
}
