在 Windows 微软视窗上挖矿

本指南将有助于在 Windows 上设置挖矿。 新的 NexusMiner 使得在 Windows 上挖矿变得非常容易。 Nexus 麟社拥有自己开放的质数矿池，开发人员正在致力于进一步分散挖矿。

下载挖矿软件：

从下面的链接下载 windows miner 可执行文件（不是安装程序）

NexusMiner.exe

注意：矿工不能在一台电脑上同时跑 prime 质数，和 hash 哈希。

Miner Configuration 矿工配置：

要配置矿工，请查看下面的链接：

  • 矿机配置 - 如何配置矿机

Nexus 麟社桌面钱包：

这仅适用于独自矿工。 矿池矿工可以跳过这一步。

麟社界面：

下载并安装 Nexus 麟社界面或设置 CLI 核心。

启动钱包，创建用户，登录并解锁钱包以进行挖矿和通知。

界面挖矿设置：

  1. 转到 settings > Core > Enable mining 通过单击旁边的切换按钮。
  2. 下面会弹出一个新字段：Mining IP Whitelist。 输入矿工的 <ipaddress:port>。 如果在同一台电脑上挖矿则输入 127.0.0.1:9325，如果矿机在另一台电脑或 FPGA 上运行则输入特定的 “ipaddress:9325。如果有多个矿机则使用 ';' 分隔 IP 地址。通配符 * 仅支持 IP 地址，例如：192.168.10.*:9325。

 

Nexus Core 麟社核心

如果使用 Nexus 麟社核心，则在每个矿工的 nexus.config 中添加一行 llpallowip=<ipaddress:port>。 使用 127.0.0.1:9325 在同一台计算机上进行挖矿，或使用 ipaddress:9325 进行个别的矿工。

重新启动核心以使更改生效

提示：要进行独自挖矿，用户必须登录并解锁才能进行挖矿和接收通知。

运行矿工 NexusMiner

转到 NexusMiner 可执行文件和 miner.conf 所在的文件夹，双击 NexusMiner 可执行文件。 将弹出一个安全警告窗口（如下图所示），单击运行，矿机将在终端中启动。 矿工将运行并开始挖矿，您可以从矿工终端窗口的消息中看到这一点。 不需要用户交互。

要检查是否一切正常，请转到下面的矿池页面链接，在标题右侧，将在 miner.conf 文件中输入的 Nexus 麟社地址粘贴到搜索框中，然后单击搜索。 这将打开如下页面，您可以在其中查看您的矿机的详细信息。

  • Prime Pool Miner Website

停止矿工 NexusMiner

要停止矿工，请关闭 NexusMiner 终端窗口。
