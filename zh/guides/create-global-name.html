创建全局名称

本指南将帮助用户使用 Nexus 麟社界面创建全局名称。 在我们开始创建全局名称之前，用户需要熟悉其概念和用法。

Global Name 全局名称:

全局名称不需要用户名称或名称空间前缀，因此是全球唯一的。 这些可能会保留给需要简洁、唯一的名称的用例，例如通证股票代码。 为避免不必要的域名抢注，全球域名收取 2000 NXS 麟社币的费用。

创建全球名称将产生 2000 NXS 麟社币的费用。 费用将从默认账户中扣除。

全局名称的用途：

全局名称的最佳用例是作为通证（代币）代码。 对于 P2P 市场，拥有通证的全球代码将非常有帮助，这将使它易于上市，也易于被用户识别。

通证可以在没有全局名称的情况下进行交易，但需要使用注册地址来寻址，不容易被用户区分因此不推荐。

Tritium++ 麟氚++:

在 Tritium++ 麟氚++ 中，当用户创建通证时，如果提供了名称参数，则默认情况下会创建一个全局名称。

 

Global Name Parameters 全局名称参数：:

创建全局名称所需的参数。

Name Field 名称字段

在此字段中，用户提供要创建的全局名称。

Register Address 注册地址

在此字段中，用户提供将链接到全局名称的通证的注册地址。

创建名称:

要使用界面创建名称，请按照以下步骤操作：

  • Open the Nexus Interface, make sure the wallet is fully synched and log into the user account (Sigchain).
  • In the Overview page, at the bottom click on the "User" module. This opens the User page.
  • In the User page, on the left side click on the "Names" tab.
  • In the Names page click on "Create a new name". This opens the Create a new name page.
  • On top of the page make sure the "GLOBAL" button is highlighted, if not click on it.
  • In this page the user has to provide the "Name" and the "register address" to which it points .
  • Once the parameters are provided, click on the "CREATE NAME" button on the bottom of the page.
  • The data will be sent to the network for verification and when the transaction is confirmed, the newly created global name will appear in the list.
