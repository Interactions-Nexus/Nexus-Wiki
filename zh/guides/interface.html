麟社界面

介绍

Nexus Interface 麟社界面是官方实施的嵌入式核心，带有一个用户界面，可以帮助普通用户访问、创建、通证、资产、发票、项目以及与其他用户进行交易。由于管理链上密钥和资产的架构不同，NXS 麟社为网络提供动力的加密货币与任何其他第三方钱包或硬件钱包并不兼容，由于管理链上密钥和资产的不同架构，类似于云服务，以及与其他区块链相比麟社具有更高的安全标准（512 位云密钥与 256 位云密钥相比）。

界面是模块化的，由核心和用户界面组成。这有助于普通用户运行界面，并使高级用户能够在服务器或无头机器上将核心作为守护进程运行，并使用界面或 API 插件进行访问。该界面允许普通用户在网络上执行各种操作，并且可以通过在安全的沙盒环境中使用模块来启用其他功能。

该接口同时支持 Tritium 麟氚和 Legacy 传统链。这是通过两种模式实现的，用户可以在麟氚模式和传统模式之间切换。

我们建议在传统链中拥有 NXS 麟社币的用户迁移到 Tritium 麟氚链。

下一个界面的升级将默认为精简节点。与今天执行的完整节点相比这将降低普通用户的复杂性。

如果任何用户遇到 Nexus 麟社界面使用上的问题，或有任何一般疑问，我们将很乐意在下面链接的 Nexus 麟社电报群上提供协助。
