介绍

Nexus Interface 是嵌入式 Core 的官方实现，带有一个用户界面，可以帮助普通用户访问、创建、令牌、资产、发票、项目以及与其他用户进行交易。由于管理链上密钥和资产的架构不同，类似于云服务，以及与其他区块链（512 位）相比更高的安全标准，为网络提供动力的加密货币 NXS 与任何其他第三方钱包或硬件钱包不兼容密钥与 256 位密钥相比）。

 

界面是模块化的，由核心和用户界面组成。这有助于普通用户运行界面，并使高级用户能够在服务器或无头机器上将核心作为守护进程运行，并使用界面或 API 进行访问。该界面允许普通用户在网络上执行各种操作，并且可以通过在安全的沙盒环境中使用模块来启用其他功能。

 

该接口同时支持 Tritium 和 Legacy 链。这是通过两种模式实现的，用户可以在氚模式和传统模式之间切换。

 

我们建议在遗留链中拥有 NXS 的用户迁移到 Tritium。

 

与今天的全节点实现相比，下一个接口更新将默认为轻节点。这将降低普通用户的复杂性。
