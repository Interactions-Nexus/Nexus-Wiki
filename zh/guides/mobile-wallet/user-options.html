用户讯息

要访问用户信息窗格，请在概览页面上点击左上角的汉堡包图标。

移动钱包 - 概览

移动钱包 - 用户信息

在用户窗格中，点击用户名称以显示用户账户选项：

  • Show user ID 显示用户身份: 这将显示用户 ID 身份。
  • Change passsword & PIN 更改登录密码和操作密码: 这将使用户能够更改登录密码和操作密码。
  • Change recovery phrase 更改复原词组: 这将使用户能够设置或更改复原词组。
  • Logout 登出: 这将允许用户登出。

在用户详细信息下方，可以找到帐户选项：

Accounts 账户: 这将为用户加载账户页面。

移动钱包 - 账户

  • Contacts 联系人: 这将加载保存在设备本地的联系人。

移动钱包 - 联系人

  • Transactions 交易: 这将加载用户帐户交易历史记录。

移动钱包 - 交易页面

Core Information 核心讯息: 这将加载核心信息页面。
