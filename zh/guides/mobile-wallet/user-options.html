移动钱包 - 用户信息

在用户窗格中，点击用户名称以显示用户账户选项：

  • Show user ID 显示用户身份: 这将显示用户 ID 身份。
  • Change passsword & PIN 更改登录密码和操作密码: 这将使用户能够更改登录密码和操作密码。
  • Change recovery phrase 更改复原词组: 这将使用户能够设置或更改复原词组。
  • Logout 登出: 这将允许用户登出。

Below the user details, find the account options :

  • Accounts: This will load the accounts page for the user.

  • Contacts: This will load the contacts, which are saved locally on the device.
