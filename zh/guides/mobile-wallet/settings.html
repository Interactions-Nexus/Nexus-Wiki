设置

要访问移动钱包的设置页面，请点击右上角的齿轮图标。 请参见下图（以红色标记）。

移动钱包 - 设置图标

设置页面选项是：

  • 应用设置
  • 核心（高级）

Application 应用

“应用程序”选项下，可以找到以下设置：

移动钱包 - 应用程序设置

  • Color scheme: 这是钱包的配色方案，可用选项为自动、浅色和深色。 选择选项。

移动钱包 - 配色方案选项

Base Currency: 这是在概览屏幕上显示余额的货币。

Hide Balances: 这将在概览屏幕上隐藏所有 NXS 麟社币，和 Token 通证余额。

移动钱包 - 隐藏余额

  • Hide unused Trust account: 此选项使用户能够从概览屏幕中隐藏 Trust 账户。 移动钱包不支持 staking 质押，这是 Trust 账户的主要功能，也是对普通用户隐藏的原因。

移动钱包 - 隐藏 Trust 账户

Core (Advanced) 核心（高级）

在核心下，找到以下选项：

移动钱包-核心设置

  • External Core: 此选项允许用户将移动钱包连接到远程核心并访问用户账户。

移动钱包外部核心设置

  • IP Address: 这是远程核心的 IP 地址。
  • API Port: 这是远程核心的 API 端口。
  • API username: 这是 API 身份验证用户名称。
  • API password: 这是 API 认证密码。
  • Core Log: 这是核心日志文件，这有助于在出现故障时诊断问题。
  • Delete Database: 这将允许用户在损坏时删除区块链数据库，这将使钱包无法正常运行。
