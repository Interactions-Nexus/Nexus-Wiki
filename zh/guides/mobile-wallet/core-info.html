核心信息

核心信息提供有关嵌入式核心和网络数据的信息。

用户可以通过两种方式访问核心信息页面：

  • 当用户未登录时：点击左上角的信息图标（标记为红色）
  • 当用户登录时：在概览中，点击左上角的汉堡包图标，打开用户信息面板，然后点击核心信息。

移动钱包概览

移动钱包 - 用户面板

Core 核心

核心信息页面如下所示： 

移动钱包核心信息

 

  • Core version 核心版本: 这是核心版本
  • LLP version 低层协议版本: 这是低层协议 (LLP) 版本
  • Host name 主机名称: 这是移动设备的主机名称
  • Timestamp 时间戳: 这是时间戳
  • Multi-user mode 多用户模式: 指示多用户状态的布尔标志。
  • Lite mode 精简模式: 布尔标志，指示移动钱包是否正在使用精简版数据库。

Network 网络

移动钱包核心信息

  • Active network 活跃网络: 这是移动钱包连接的网络。
  • Connections 连接: 移动钱包的对等连接总数。
  • Synchronizing 同步：用于同步的布尔标志。
  • Sync. progress 同步进展: 同步百分比。
  • Block height 区块高度: 移动钱包当前的区块高度。
