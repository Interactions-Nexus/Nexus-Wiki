发送 NXS 麟社币，或 Token 通证

要将 NXS 麟社币或 Token 通证发送到另一个账户，请在钱包概览页面上，点击特定 NXS 麟社币或 Token 通证账户的发送。

移动钱包 - 概览

这将打开发送 NXS 麟社币屏幕。

移动钱包 - 发送

SEND FROM 发送自

这里列出的是要发送 NXS 麟社币，或 Token 通证的账户，请检查发送 NXS 麟社币的账户地址是否正确。
 

SEND TO 发送至

Recipient 接受者：输入接受者的用户名称或地址。 用户名称必须采用 Zin:default 格式（ZIN 是用户名称，default 是账户）。 在接受者输入框下方有三个图标，用户可以使用它们从剪贴板粘贴、从联系人中选择或扫描二维码。

Amount 数额: 输入要发送的 NXS 麟社币或 Token 通证代币数额，如果要发送全部可用余额，请在数额输入框下选择发送全部。

在高级选项下：

  • Reference Number 参考号码: 这是一个用于识别交易的数字，例如发票号码或订单号码。 该引用是一个 64 位无符号整数，范围为 0 到 18446744073709551615。

移动钱包 - 继续发送

输入所有信息后，点击继续按钮或图标。 这会将交易发送到网络进行验证，然后将其包含在区块链中。
