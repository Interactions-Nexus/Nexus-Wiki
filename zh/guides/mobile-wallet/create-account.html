创建账户

这将引导用户创建一个新的 NXS 麟社币或 Token 通证账户。 用户需要有通证的全局名称或通证地址才能创建通证账户。

每个用户账户或配置文件都必须为用户想要交易的每个通证创建单独的通证账户。

创建 NXS 麟社币账户

要创建新的 NXS 麟社币账户，请在 “概览” 页面上点击打开用户窗格的汉堡包图标，点击 “账户” 以打开 “账户” 页面，然后点击底部的 “添加” 图标以打开 “新账户” 页面。

移动钱包 - 用户账户

移动钱包 - 选择 NXS 麟社币， 或 token 通证

输入账户名称，这是有选择性的，费用为 1 NXS 麟社币，检查账户通证是否显示 NXS 麟社币。 点击创建账户以创建新的 NXS 麟社币账户。 这将返回到列出新账户的页面。

创建 Token 通证账户

要创建一个新的通证账户，有点不同，在 “新账户” 页面中，点击账户通证 NXS，这会打开一个弹出式选择通证。 在搜索框中输入用户想要创建的通证的全局名称或地址。 点击搜索。

移动钱包 - 选择 NXS 麟社币， 或 token 通证

这将打开带有通证地址的新账户页面。

在账户名称字段中输入一个名称，这是有选择性的，费用为 1 NXS 麟社币。 点击创建账户以创建通证账户。 这将返回到列出新账户的页面。
