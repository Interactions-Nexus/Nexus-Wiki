开始质押

将质押币发送到 trust 账户。（所有不要质押的 NXS 麟社币保留在 default 默认账户中）。

NXS 麟社币需要在 truat 账户中老化 72 小时才能开始质押。 如果在 72 小时老化窗口之间增加任何额外的麟社币，计时器将重置为开始。

72 小时后，钱包会尝试找到一个区块，并根据账户中 NXS 麟社币的数量启动 Genesis 起源交易，将质押金额锁定在 trust 账户中。 Genesis 起源是您质押之旅的开始。

要将激励从 0.5% 提升到 3%（权益利率），钱包必须在前一个质押区块的 72 小时内质押一个区块（开采一个区块）。 （区块重量 = 100%）。

质押必须持续进行，才能将权益利率提高到 3%； 这是在持续质押一年后实现的，此后要保持它。

获得的奖励将累积在 “可用余额” 下的 trust 账户中，不会添加到质押中。 应计奖励可以提取而不会受到处罚。

由于节点大部分时间可能自己没看管，请至少每隔几天保持警惕并验证活动，这也可以通过区块浏览器来完成。 请记住，如果节点出于任何原因停止质押，信誉和权益利率将以 3:1 的比例衰减。
