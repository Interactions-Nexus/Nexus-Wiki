质押要求

操作系统:

支持所有主要操作系统 (OS)，但社群推荐的操作系统是 Ubuntu LTS 或衍生版本。 可以使用任何 Linux 发行版，但指南是为 Ubuntu 量身定制的。

计算机要求：

该守护程序经过测试，可以在具有 1GB Ram 的 Raspberry Pi 3B+ 上完美运行，但最低建议如下，以预备往后的增长：

ARM:

具有 2GB RAM 的 Raspberry Pi 4B 或具有最小 64 GB SD 卡的 2GB 等效 SBC（仅限 Linux 上的守护程序/CLI）。

x64:

任何具有 4GB RAM 的处理器。 在安装钱包之前，请确保您至少有 30GB 的可用硬盘空间。

注意：完整节点将下载区块链数据库的完整副本，截至今天，提取的数据库大小约为 16 GB，并且会随着时间的推移而增长。 精简节点将仅维护区块头和签名链数据，从而极大地减少存储影响，大约 250MB。

模块化钱包设计：

Nexus 麟社钱包包括由接口、模块框架和核心组成的模块化功能。 直观的界面为用户和企业提供了轻松无缝的体验。 节点也可以作为守护程序专门运行，并通过 API 或远程接口进行管理。 当在 Raspberry Pi 或 NUC 等无头系统上运行质押节点时，这是有利的。

经济实施和物联网 IoT：

PoS 的基本原则是保护网络，并为此提供激励。 节点运营者承担硬件、管理、带宽和电力方面的费用。 本节中的说明为严格的预算和 IoT 物联网行业提供了最佳配置。

一般用户（GUI钱包）：

Any computer which meets the minimum requirements, If staking is the only service planned an Intel NUC core i3 / Ryzen 3 or equivalent with 4GB RAM and 120GB SSD is recommended. If the staking computer will be used for other purposes, increase memory and disk space appropriately.

Intel NUC or equivalent will be a good choice if looking for standalone staking node

Advanced Users (CLI Wallet):

Raspberry Pi (RPI) 4 with 2/4GB, NUC or equivalent will run the headless core and users can control it via the interface from another computer. (RPI can run only the CLI daemon so command line interaction is necessary. Nexus has great step-by-step guides to aid those who are unfamiliar.)

The core is very light on resources even on a Raspberry Pi
