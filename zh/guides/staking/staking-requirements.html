质押要求

操作系统:

支持所有主要操作系统 (OS)，但社群推荐的操作系统是 Ubuntu LTS 或衍生版本。 可以使用任何 Linux 发行版，但指南是为 Ubuntu 量身定制的。

计算机要求：

该守护程序经过测试，可以在具有 1GB Ram 的 Raspberry Pi 3B+ 上完美运行，但最低建议如下，以预备往后的增长：

ARM:

具有 2GB RAM 的 Raspberry Pi 4B 或具有最小 64 GB SD 卡的 2GB 等效 SBC（仅限 Linux 上的守护程序/CLI）。

x64:

任何具有 4GB RAM 的处理器。 在安装钱包之前，请确保您至少有 30GB 的可用硬盘空间。

注意：完整节点将下载区块链数据库的完整副本，截至今天，提取的数据库大小约为 16 GB，并且会随着时间的推移而增长。 精简节点将仅维护区块头和签名链数据，从而极大地减少存储影响，大约 250MB。

模块化钱包设计：

Nexus 麟社钱包包括由接口、模块框架和核心组成的模块化功能。 直观的界面为用户和企业提供了轻松无缝的体验。 节点也可以作为守护程序专门运行，并通过 API 应用程式界面或远程接口进行管理。 当在 Raspberry Pi 或 NUC 等无头系统上运行质押节点时，这是有利的。

经济实施和物联网 IoT：

PoS 的基本原则是保护网络，并为此提供激励。 节点运营者承担硬件、管理、带宽和电力方面的费用。 本节中的说明为严格的预算和 IoT 物联网行业提供了最佳配置。

一般用户（GUI钱包）：

任何满足最低要求的计算机，如果质押是计划的唯一服务，建议使用 Intel NUC core i3 / Ryzen 3 或具有 4GB RAM 和 120GB SSD 的同等产品。 如果质押电脑将用于其他用途，请适当增加内存和磁盘空间。

如果寻找独立的质押节点，英特尔 NUC 或同等产品将是一个不错的选择

高级用户（CLI 钱包）：

具有 2/4GB、NUC 或同等设备的 Raspberry Pi (RPI) 4 将运行无头内核，用户可以通过另一台计算机的界面对其进行控制。 （RPI 只能运行 CLI 守护程序，因此命令行交互是必要的。Nexus 麟社有很好的分步指南来帮助那些不熟悉的人。）

即使在 Raspberry Pi 上，内核也非常占用资源。
