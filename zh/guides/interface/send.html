发送

要将 NXS 麟社币或通证发送到另一个地址，或在您的个人帐户之间移动 NXS 麟社币，请转到发送模块，并按照以下说明进行操作：

1. 发送自：从下拉列表选择钱包可用的要发送金额的账户。

2. 发送至：从下拉列表选择您的其他收款帐户之一，或通讯录中保存的收款联系人，也可以复制并粘贴新的收款人地址。 Tritium 麟氚允许使用接收方用，用户名称:帐户 username:account（区分大小写），如下图 Zin:default（ZIN 是用户名称，default 是帐户），新手使用用户名称可以轻松进行交易，也可以避免在复制时出错 十六进制地址。

3. Amount 数额：输入您要发送的 NXS 麟社币或通证数量。

4. Reference Number 参考号码：选择性要不要添加参考编号、发票号码或订单号码。

要发送通证，请使用下拉列表在 “发送自” 字段中选择该帐户。 这会将使数额标签更改为所选通证。
