交易

交易模块显示当前登录的签名链（SigChain）上的所有交易。

可以按以下任何条件过滤交易：

⇢ Address 地址：指定 SigChain 签名链上以前用于发送或接收 NXS 麟社币或通证的地址。

⇢ Name 名称：指定 “NXS” 麟社币或之前参与此 SigChain 交易的通证名称。

⇢ Time Span 时间跨度：使用下图中的时间段之一，指定要过滤的交易的时间段。

⇢ Operation 操作：使用下图中的操作名称之一，指定要过滤的交易操作。

⇢ Write 写入：用于将原始数据写入寄存器。

⇢ Append 追加：应用于仅追加寄存器的附加数据。

⇢ Create 创建：用于创建新的对象寄存器（例如帐户、资产、通证、名称）。

⇢ Transfer 转移：将寄存器（例如资产）的持有权转移到另一个签名链。

⇢ Claim 声明：声明对已转让给您的寄存器的持有权。

⇢ Coinbase 币库：矿工奖励交易。

⇢ Trust 信誉：质押奖励交易。

⇢ Genesis 起源：第一笔质押交易初始化质押过程。

⇢ Debit 借记：从帐户发送的 NXS/通证。

⇢ Credit 贷记：NXS/代币收到一个帐户。

⇢ Migrate 迁移：将传统链里的 trust 密钥一次性迁移到 Tritium staking 麟氚链质押。

⇢ Fee 费用：交易的相关费用。

⇢ Lagacy 传统链：从传统链地址进行或接收到传统链地址的交易。

单击任何交易以查看更多详细信息，例如帐户地址、交易身份证、哈希、确认次数、通证名称和交易数额。
