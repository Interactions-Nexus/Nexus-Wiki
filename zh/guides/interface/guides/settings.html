设置

设置中有 4 个页面：Application 应用、Core 核心、Style 样式、Modules 模块。

 

Application 应用

Language 语言: 选择您选择的语言。 默认语言为英语。

Minimize on close 关闭时最小化: 启用此功能后，即使钱包关闭，它仍会在后台打开。 要重新打开您的 Nexus 麟社钱包，请执行以下操作：

  • Mac 苹果机: 点击右上角的 Nexus 麟社标志，然后点击 “显示 Nexus 麟社钱包”
  • Windows 视窗机: 双击右下角菜单中的 Nexus 麟社标识。

启用此功能后，您的钱包（节点）仍将在线同步； 因此，这是比较方便的。 如果您的钱包应用程序完全关闭（关闭时最小化设置已禁用），那么当您下次打开该应用程序时，您将不会与 Nexus 区块链同步，因此需要跟上当前区块。

Auto update 自动更新: 此选择可以在无在每次发布更新时下载新钱包新版本（这是许多加密货币钱包的必要功能），而是会通知您要接受的更新，这与常规计算机软件更新相当。

Nexus 麟社强烈建议此设置保持启用状态。 启用此设置将自动检查新版本并在有新版本可用时通知您。 如果未启用自动更新，您将需要从 github.com/Nexusoft/NexusInterface 或我们的网站手动下载新的钱包版本。

任何钱包更新都将显示在弹出框中。 必须尽快安装这些更新。 如果您的钱包没有更新到最新版本，您的密币可能面临风险，并且可能意味着您不再与 Nexus 区块链同步。 请确保在收到更新通知后立即更新您的电子钱包。

Send anonymous usage data 发送匿名使用数据: 发送匿名数据使用以允许 Nexus 麟社开发人员改进钱包。

Base Currency 基准货币: 选项将更改显示余额、市场价格、市值和 24 小时的货币单位。默认货币是美元。

Minimum Confirmations 最低确认: 接受块之前的最低确认区块次数。 最低确认次数为 1，但您可以自定义在钱包将交易视为有效之前需要等待的次数。 建议进行 3 到 6 次确认以保持高级别的安全性。

Backup Directory 备份目录: 这是存储您的 Nexus 麟社钱包备份的地方。 当您单击 “备份钱包” 时，系统会提示您选择一个自定义目录，而不是默认的 “Nexus Backups” 目录。

Developer Mode 开发模式: 开发模式启用高级功能以帮助开发。 启用后，必须关闭并重新打开钱包才能启用这些功能。

 

Core 核心

Enable Mining 启用挖矿: 启用挖矿后，您的钱包将运行一个名为 “Mining LLP” 的特殊服务器，它将允许您将外部矿工连接到您的钱包，以便在 Prime 质数或 Hash 哈希通道上生产区块。

Enable Staking 启用质押: 启用质押后（默认设置），您的钱包将启用质押，并会在您登录时对您的信誉账户进行质押。您还必须启用 “Staking Enabled” 启用质押才能让您的钱包接收信誉收益。 这是为了防止人们无意中启动质押。

Verbose Level 详细级别: 日志的详细级别显示了网络上更多的幕后数据。 如果您不需要查看大量日志记录信息，我们建议您运行 verbose level 0。 如果您遇到钱包问题，更多的调试数据对于开发人员用来查找错误和推送新版本很重要。 请自行决定使用此设置。

 

Style 样式

Render Globe 渲染地球仪: 打开或关闭地球仪。

Overview Display 概览显示: 自定义钱包显示。 选项为：

  • Standard 标准: 钱包随地球仪一起。
  • Miner 矿工: 显示挖矿指标，例如 Prime 质数、Hash 哈希和 Stake 质押难度。
  • Hidden Balance 隐藏余额: 隐藏您的 NXS 麟社币、美元和收款余额。
  • None 无: 仅显示模块和闪烁的星星。

Nexus Address Format 麟社地址格式: 根据您的喜好格式化您的 NXS 麟社币地址。 请注意，复制地址时不会复制格式（例如分段样式中的 9 个空格）。 这仅用于视觉偏好。 选项是：

  • Segmented 分段样式
  • Truncate middle 截断中间
  • Raw 原始

 

Theme 主题：:

有两个默认主题 Dark 黑色和 Light 浅色。

Background 背景: We provide two default default backgrounds Twinkling stars and Cosmic Light. You can also upload a custom wallpaper that you have saved on your device.

Color scheme: Change the colors to customize your own theme. Upon changing a color in one of the default themes you will see that your ‘Theme’ changes from either Dark or Light to ‘Custom theme’. If you were to then revert back from Custom theme to Dark theme for example, and then back to ‘Custom theme’, you will find that your custom theme has been saved.

Import Custom Theme: A variety of custom themes created by the Nexus community can be imported into the Wallet. The below link holds a repository of community made themes. The background can be imported from your local machine, or enter a URL and the Wallet will download the background for you. Acceptable formats are PNG/JPEG/BMP/TIFF/GIF.

Export Custom Theme: Themes are Json files that are read by the system and apply changes to the interface. When you export a theme, you will be prompted to save the json file to your computer. Below is a link to the theme guide which describes how to develop themes. If you would like to, please request that your theme is included in the official repository for it to be shared with the community.

Modules

Security is very important, and therefore modules are sandboxed to prevent any misuse.

To import a module, simply drag the file into the center of this page.

Module app icons will appear in the bottom menu to the right of the Console icon.

Documentation on how to write modules : https://github.com/Nexusoft/NexusInterface/tree/master/docs/Modules

We recommend using one of these links as a starting point:

  • React-Redux Example: https://github.com/Nexusoft/react_redux_module_example
  • React Example: https://github.com/Nexusoft/simple_react_module_example
  • HTML Example: https://github.com/Nexusoft/minimal_module_example
