创建用户帐户

在创建新用户账户之前，钱包必须与 Nexus 网络完全同步。 如果这仍然需要完成，请参阅引导和同步。

 

要创建您的用户帐户，只需选择用户名、密码和 PIN。

 

{% hint style="info" %}

Nexus 上的用户帐户也称为签名链 (SigChains)

 

用户名：长度至少为3个字符，区分大小写。 示例：Oliver Queen、Oliver.Queen、oliverqueen。 我们建议保持用户名简单，因为这是在 Tritium 上进行交易的最简单方式（用户名：default 将是接收 nxs 或令牌的地址。示例：Oliver Queen：default）。

 

密码：必须至少为 8 个字符。 它可以包含大小写字母、0-9 的数字和特殊字符。

 

PIN：必须至少为 4 位数字。 它可以包含字母数字和特殊字符，并且可以用作第二个密码。
