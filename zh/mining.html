挖矿

Nexus 麟联社有三个共识通道，两个挖 hash 哈希，prime 质数，第三个是 stake 质押。 任何人都可以在 Nexus 麟社上挖矿，不需要许可。 Nexus 麟社开发团队有一个专门的挖矿开发团队来维护矿工和矿池。

挖矿支援：

如需与挖矿相关的任何帮助或支援，请加入我们下面链接的电报频道：

NexusMiners

挖矿通道：

  • Prime Mining 质数挖矿: GPU 显卡- 支持 Solo 独自和 Pool 矿池。
  • Hash Mining 哈希挖矿: GPU 显卡 or FPGA’s 现场可编程门阵列 - 支持 Solo 独自和 Pool 矿池。

提示：我们不推荐以 CPU 质数或哈希挖掘，因为它无利可图且仅用于测试网挖掘。

Solo Mining 独自挖矿:

Solo 是独自或个人挖矿。 它需要矿工连接到 Nexus 麟社钱包。 钱包和矿工可以在同一台或单独的计算机上运行，尤其是在使用 FPGA 现场可编程门阵列进行挖掘时。

提示：独自挖矿对 prime 质数和 hash 哈希来说是无利可图的，除非你有一个非常大的挖矿设备。

Pool Mining 矿池挖矿

矿池挖矿类似于群挖矿，矿机必须接入矿池。 矿工直连矿池，挖矿奖励根据个人算力的百分比支付给矿工。 在矿池上挖矿需要支付费用，这些费用会从挖矿支出中扣除。

兼容的挖矿硬件

NexusMiner 可以使用 CPU 中央处理器、GPU 显卡进行 Prime 质数和 FPGA 现场可编程门阵列进行哈希挖掘。 不推荐使用 CPU 中央处理器挖矿，因为它效率不高。

GPU 显卡:

如果使用 GPU 挖掘，NexusMIner V1.5 支持 nvidia 和 AMD GPU 显卡。 来自 Nvidia 的 CUDA 核心得到全面支持并高效工作。 AMD 支持最近仅在 linux 上启用了 prime，并且需要一些时间才能完全有效。

提示：AMD GPU 显卡支持仍处于早期阶段，未来会得到改进

为了充分利用 nvidia 显卡，我们推荐使用最新的显卡驱动程序和  MSI Afterburner

不要安装 Nvidia content creator 驱动，矿机将无法正常工作。

FPGA 现场可编程门阵列:

FPGA 现场可编程门阵列 Nexus 麟社的矿工只能从 Blackminer 获得。 这些矿机只能用于哈希挖矿。
