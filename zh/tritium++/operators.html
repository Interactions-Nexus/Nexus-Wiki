运算符

Nexus API 麟社应用程式界面和相应的命令支持不同类型的运算符。 运算符允许在过滤后的数据集上执行计算，就像在 Excel 电子表格中一样。

运算符 URI 的最低要求组件是：

commands/verb/noun/filter/operator

Filter 过滤器

为了执行一个运算符，必须对数据集应用一个基本过滤器，因为我们需要隔离返回数据的一个组成部分以进行操作并返回。

要对我们的 amount 数额字段进行操作，我们首先需要应用一个过滤器：

commands/verb/noun/amount

此命令现在将为运算符做准备，可以通过完成 URI 来调用它：

commands/verb/noun/amount/sum

上面的命令将返回一个包含我们过滤结果的 JSON 物件：

{
    "amount": 414.53867,
}

Operators 运算符

此 API 应用程式界面命令集支持以下运算符：

array 数组 - 将一组过滤结果中给定的值列表生成为 JSON 数组。

count 计数 - 查找给定 JSON 列表中返回的条目总数。

floor 地面 - 通过去除小数将结果中的所有值转换为整数。

max 最大值 - 找到给定集合中的最大值，并将其作为结果返回。

mean 平均值 - 计算一组过滤结果的平均值。

min 最小值 - 在给定的集合中找到最小值，并将其作为结果返回。

mode - Find the value with the most occurrences across the data-set.

sum - Compute a sum of a set of values derived from filtered results.

Example:

finance/list/accounts/balance/sum

Result:

This command will return a sum of the balances for all accounts:

{
    "balance": 333.376
}
