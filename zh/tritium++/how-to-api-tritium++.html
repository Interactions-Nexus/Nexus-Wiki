Nexus 麟社 API 應用程式介面

访问 Nexus 麟社 API 應用程式介面是一种无缝且简单的体验。 要访问 API 應用程式介面，您可以使用以下三个选项之一：
 

使用基于 URL 的请求的网络浏览器：

使用 URL 中的参数发出 GET 请求，如下所示。

http://localhost:8080/<api>/<method>?<key>=<value>&<key1>=<value1>

API 應用程式介面使用表单编码，因此您可以随意使用任何字符。 唯一需要注意的是在编码中使用 “+” 字符时，它充当空格和 “%20” 的转义字符。 只要表单在表单编码中对 + 符号进行编码，API 應用程式介面就会检测到此行为。 有时发出 GET 请求，Web 浏览器会假定 + 是一个空格而不对其进行编码，发出 GET 请求时 '%' 也是如此。
 

使用带有基于 POST 的请求的 Web 浏览器

POST 正文包含参数，并将转到一个端点，例如：

http://localhost:8080/<api>/<method>

一个示例 Web 表单如下所示：

<form action='http://localhost:8080/<api>/<method>' method='post'>
<input type = 'text' name = '<key>'  value = '<value>' >
<input type = 'text' name = '<key1>' value = '<value2>'>
<input type = 'submit'>
</form>

这将允许您将 Nexus API 麟社應用程式介面集成到现有的 Web 表单中。 这使您可以在所有现有 Web 服务中利用 Nexus 麟社区块链的强大功能。
 

使用 nexus 麟社命令行：

Nexus Daemon 麟社守护程序必须已经在运行才能工作。 确保您与 Nexus 守护程序位于同一目录中。

./nexus <api>/<method> <key>=<value> <key1>=<value1>

这会将 JSON 格式的所有数据返回到您的控制台。
 

Results JSON

您将收到的 JSON 回复将包含两个键、results 或 error 之一。 成功执行命令后，您将收到一个 JSON 字符串，例如：

{"result":{"genesis":"69c2479c6780782448f419a0865542002ee85fec39228275b2db44bb6d3aa503","session":4940881975319897416}}

您将需要用您选择的相应语言解析结果物件的值。 如果您碰巧收到失败消息，它将如下所示：

{"error":{"code":-24,"message":"Missing Password"}}

请注意，您现在将解析错误键，并且不会有任何结果。 这允许您作为程序员选择检查 “error” 键是否存在以检查错误，或检查 “result” 键是否存在以查找是否执行成功。 你可以选择如何处理它，这是一个非此即彼的情况。 你只会得到一个或另一个。

 

Profiles API 配置應用程式介面

Profiles API 配置應用程式介面负责维护用户帐户级别的功能。 不要在您不是其管理者的外部节点上使用此 API 應用程式介面。 您的用户名和密码存储在安全分配器中，但远程端点仍然可以在您提交 API 應用程式介面调用时收集您的凭据。 此 API 應用程式介面仅适用于您控制的服务节点。

Tokens API 通证應用程式介面

Tokens API 通证應用程式介面允许您创建通证和账户，以便相互发送和接收通证。 这也是创建旨在成为数字资产或公司 (STO) 股份的通证的先决条件。

Ledger API 分类账本應用程式介面

Ledger  API 分类账本應用程式介面提供对分类账本的访问，主要用于创建和检索块和交易，但也用于查询分类账本状态信息。

 

Supply API 供应應用程式介面

Supply API 供应應用程式介面负责处理供应链物流。 它的主要目的是设计用于管理供应链上的路线和移动，以及审查与保管权变更相关的事件历史。
 

注意：此 API 應用程式介面中的某些命令需要登录。 在使用 LOGIN 必需命令之前，请确保使用用户 API 應用程式介面登录。

Invoices API 发票應用程式介面

Invoices API 发票應用程式介面为用户和应用程序开发人员提供了在 Nexus 麟社区块链上开具和支付发票的能力。 该 API 應用程式介面演示了如何使用寄存器存储任意用户数据，以及如何使用条件契约来实施用例规则。 可以创建发票，然后将其发送到收件人签名链进行支付。 API 发票應用程式介面支持以 NXS 麟社币或 Nexus 麟社区块链上的任何其他通证开具的发票。
