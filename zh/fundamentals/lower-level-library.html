低层库 (LLL)

低层库 (LLL) 是 TAO (麟道）框架的基础。它是一个用 C++ 编写的基本模板库，可最大限度地减少匹配硬件，且最大限度地提高裸机效率。在架构上，LLL 低层是一种可互换的结构，需要为特定功能逻辑开发的模板和模块。 LLL-TAO 或 TAO 框架是一系列 LLL 低层库模板和数据模型，可通过简单的基于 JSON 的 API 应用插件访问，允许任何类型的开发人员提高其应用程序的安全性、可扩展性和稳固性。

LLL 底层库包含三个主要组件：低层加密 (LLC)、数据库 (LLD) 和协议 (LLP)。堆栈中有几种 LLD 数据库表示形式；分类帐本、注册、操作和 API 应用插件。 LLC 主要应用于分类帐层面，尽管可以应用在其他地方。作为网络层的一个组件，LLP 被设计为一种轻便、快速的协议，允许开发人员自定义他们的数据包设计和信息注解。
